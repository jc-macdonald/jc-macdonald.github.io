projects

Research software for scientific computing, Bayesian inference, and operational modeling infrastructure.

lead developer · released

vbpca-py

Variational Bayesian PCA for incomplete data with native missing-data handling, uncertainty quantification, and C++-accelerated kernels. scikit-learn compatible.

lead developer · in development

OP Engine

Operator-partitioned solver for composing heterogeneous subsystem solvers (ODE, PDE, stochastic, hybrid) into a single time-stepping pipeline with pluggable backends.

trade-study

Multi-objective trade-study orchestration with Pareto optimization, proper scoring rules, and Bayesian stacking for scientific model criticism.

OP System

Declarative governing equation specification compiler for structured dynamical systems. Transforms YAML model definitions into callable numerics.

pp-eigentest

Posterior predictive eigenvalue testing for determining signal rank in covariance and Gram matrices. Supports ordered hypothesis testing with FWER and FDR control.

collaborative infrastructure

FlepiMoP2

Modular epidemic modeling and simulation pipeline for forecasting and scenario analysis. Core contributor to the redesign of ACCIDDA's operational infrastructure.
